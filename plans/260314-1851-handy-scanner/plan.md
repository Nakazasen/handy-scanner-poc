Created: 2026-03-14T18:52:00

## Overview
Ứng dụng Android chạy trên Handy Terminal công nghiệp dùng Google ML Kit nhận diện siêu tốc mã QR (2D) và Code 39 (1D) với âm thanh báo hiệu. Toàn bộ logic build tự động file APK qua GitHub Actions do máy client không cài Android Studio.

## Tech Stack
- Frontend: Android XML Layout
- Backend (Logic): Kotlin + CameraX + ML Kit
- CI/CD: GitHub Actions (Gradle Build)
